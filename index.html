Tag: <h1>
É uma tag utilizada para títulos
Tag: <p>
É uma tag utilizada para parágrafos
Tag: <img>
É uma tag que busca uma imagem por um caminho através do src="" e imprime ao usuário
Tag: <input>
É uma tag em que permite ao usuário preencher dados de entrada
Tag: <div>
É uma tag utilizada para agrupar elementos de um código html
Tag: <br>
É uma tag utilizada para realizar quebra de linhas
Tag: <hr>
É uma tag utilizada criar linhas de divisão
Tag: <nav>
É uma tag utilizada para criar uma barra de navegação
Tag: <style>
É uma tag específica para inserir CSS no arquivo
Tag: <strong>
É uma tag destinada a descrever palavras em negrito
